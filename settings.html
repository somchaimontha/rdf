สิทธิ์ผู้ใช้

ตาราง Permission Matrix และกำหนดสิทธิ์นักเรียน

ระบบสิทธิ์ผู้ใช้งาน (Role-Based Access Control)

ผู้ดูแลระบบแต่ละประเภทมีสิทธิ์เข้าถึงข้อมูลและฟังก์ชันที่แตกต่างกัน

SuperAdmin

สิทธิ์สูงสุด
  • จัดการผู้ดูแลระบบทั้งหมด
  • ตั้งค่าระบบ, API Keys, สาขาวิชา
  • เพิ่ม / แก้ไข / ลบนักเรียน
  • ดูประวัติการใช้งานทั้งหมด
  • จัดการเซสชัน / ผู้ใช้งาน

Manager

ผู้จัดการ
  • เพิ่ม / แก้ไข / ลบนักเรียน
  • ดูข้อมูลนักเรียนทั้งหมด
  • พิมพ์โปรไฟล์และรายงาน
  • นำเข้า CSV / จัดการข้อมูล
  • ตั้งค่าระบบ / จัดการ Admin

Finance

การเงิน
  • ดูข้อมูลนักเรียนทั้งหมด
  • ดูข้อมูลการเงิน / ทุนการศึกษา
  • พิมพ์รายงานการเงิน
  • แก้ไขข้อมูลนักเรียน
  • ตั้งค่าระบบ / จัดการ Admin

Committee

คณะกรรมการ
  • ดูข้อมูลนักเรียนทั้งหมด
  • พิมพ์โปรไฟล์นักเรียน
  • ดูสถิติและภาพรวมระบบ
  • แก้ไขข้อมูลนักเรียน
  • ตั้งค่าระบบ / จัดการ Admin

DormTeacher

ครูหอพัก
  • ดูข้อมูลนักเรียน (เฉพาะ MBS/VC)
  • ข้อมูลติดต่อ / ผู้ปกครอง
  • แก้ไขข้อมูลเบื้องต้น
  • ดูข้อมูลการเงิน
  • ตั้งค่าระบบ / จัดการ Admin

Student

นักเรียนทุน
  • ดูข้อมูลโปรไฟล์ตนเอง
  • อัปเดตข้อมูลส่วนตัว
  • อัปโหลดรูปโปรไฟล์
  • ดูข้อมูลนักเรียนคนอื่น
  • ตั้งค่าระบบ
ตาราง Permission Matrix คลิกเซลล์เพื่อแก้ไข

SuperAdmin (สิทธิ์สูงสุดทุกอย่าง) และ Student (เฉพาะโปรไฟล์ตนเอง) ถูกล็อคไว้เสมอ — ไม่สามารถแก้ไขได้
Roles ที่แก้ไขได้: Manager, Finance, Committee, DormTeacher
